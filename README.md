# Vibrant Calculator

A modern, high-performance scientific calculator for Android. Built with a focus on a vibrant aesthetic and streamlined user experience, this app provides essential arithmetic and trigonometric functions with a sleek Material Design 3 interface.

## Features

*   **Sequential Arithmetic Engine**: Performs addition, subtraction, multiplication, and division using a straightforward left-to-right execution logic.
*   **Scientific Functions**: Supports Sine, Cosine, and Tangent operations with a dedicated toggle for Degrees and Radians.
*   **Modern UI**: Fully immersive edge-to-edge display using Jetpack Compose and Material Design 3.
*   **Responsive Design**: Utilizes ConstraintLayout for Compose to ensure a perfect fit across different screen sizes.
*   **Adaptive Icon**: A custom-designed adaptive icon that fits seamlessly with the Android ecosystem.

## Tech Stack

*   **Language**: Kotlin
*   **UI Framework**: Jetpack Compose
*   **Design System**: Material Design 3
*   **Architecture**: MVVM (Model-View-ViewModel)
*   **State Management**: Kotlin Flow and StateFlow
*   **Concurrency**: Kotlin Coroutines

## Getting Started

1.  Clone the repository.
2.  Open the project in Android Studio (Ladybug or newer recommended).
3.  Build and run on an emulator or physical device running Android 10 (API 29) or higher.
